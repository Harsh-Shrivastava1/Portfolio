SplitChain

SplitChain is an advanced on-chain group expense splitter designed for seamless and secure expense management. Built using HTML, CSS, and TypeScript, it allows users to create groups, track expenses, and settle payments using USDC through Coinbase Wallet integration. The platform offers a clean, user-friendly interface with localStorage for efficient data management, ensuring that user data remains accessible even offline. SplitChain is a secure and scalable solution that leverages blockchain technology for transparent and tamper-proof expense tracking, making it a standout project developed for the Base India Hackathon.
